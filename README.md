# QR Code Generator

![Screenshot](./Screenshot.png)

# What is it?

This is a website for **generating QR Codes**. You can customize the **colors** and **link** of the QR Code. You can
also **download** it in **PNG** format.

# Quick Start

The website can be accessed at https://free-qrcode-generatorr.netlify.app

# Source Code

Clone the repository:

```
git clone https://github.com/sh4man4ik/QRCodeGenerator.git
```

Go to the directory:

```
cd QRCodeGenerator
```

Download the dependencies:

```
npm install
```

Launch the website:

```
npm run dev
```

# Technology

The following technologies are used to develop the website: _TypeScript_, _HTML_, _CSS_, _Vite_, _Google Fonts_ and
_qrcode.js_.

# Facts about the website

- To monetize the project, the website uses **non-intrusive advertising**
- Google Analytics is used to view **stats**
- The website is **responsive**
